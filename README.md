# 🎯 BinaryQuest: O Jogo de Adivinhação

Um jogo interativo de console desenvolvido em Python onde o desafio é descobrir três números secretos gerados aleatoriamente com o menor número de tentativas possível.

## 🚀 Sobre o Projeto

Este projeto foi criado para exercitar conceitos fundamentais de programação, como estruturas de repetição, condicionais e manipulação de entradas do usuário. O jogo é dividido em 3 rodadas consecutivas, e ao final, o jogador recebe um feedback do seu desempenho total.

### 🧠 Lógica Aplicada
O jogo utiliza:
* **Geração Aleatória:** Uso da biblioteca `random` para garantir que cada partida seja única.
* **Laços de Repetição:** `for` para controlar as rodadas e `while` para validar os palpites.
* **Fluxo de Controle:** Comparação de magnitude (maior/menor) para guiar o usuário até a resposta correta.
